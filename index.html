TCP Port Scanner

socket networking nmap

A Python TCP port scanner that connects to each port on a target host to check its state. The script is fast, freeing up resources quickly so you can take action elsewhere in the security chain.

Initial test on 127.0.0.1 reported 0 open ports. After correcting a grammatical error in the script, a clean dry run against nmap.org returned 2 open ports: 22 (SSH) and 80 (HTTP).

Python · port_scanner.py 
import socket
import sys

def scan_ports(host, start_port=1, end_port=1024):
    """TCP port scanner — checks each port for an open connection."""
    open_ports = []
    for port in range(start_port, end_port + 1):
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.settimeout(0.5)
        result = sock.connect_ex((host, port))
        if result == 0:
            open_ports.append(port)
            print(f"  ✓ Port {port} OPEN")
        sock.close()
    return open_ports

if __name__ == "__main__":
    target = sys.argv[1] if len(sys.argv) > 1 else "127.0.0.1"
    scan_ports(target)
✅
Terminal Output — Clean run against nmap.org returned port 22 (SSH) and port 80 (HTTP) as open. Earlier run on 127.0.0.1 returned 0 open ports. Grammatical errors in an earlier version were corrected for a successful run — the full terminal input/output is documented in the project repo.

Automated Log Analysis

re collections SSH brute-force detection

Parses auth.log for failed SSH login attempts and flags any IP address that exceeds a configurable threshold — a key pattern in detecting brute-force attacks.

A test file of synthetic log data was created to run the script against. 192.168.1.100 was correctly flagged as the attacker — it logged 3 failed attempts, triggering the rule that flags anything above the threshold of 2. The second IP was not flagged because it only registered 1 attempt.

Python · log_analyzer.py 
import re
from collections import Counter

def analyze_auth_log(filepath, threshold=10):
    """Parse auth.log for failed SSH attempts and flag suspicious IPs."""
    failed_pattern = re.compile(
        r"Failed password for (?:invalid user )?(\S+) from (\d+\.\d+\.\d+\.\d+)"
    )
    attempts = []
    usernames_targeted = {}

    with open(filepath, "r") as log_file:
        for line in log_file:
            match = failed_pattern.search(line)
            if match:
                username  = match.group(1)
                ip_address = match.group(2)
                attempts.append(ip_address)
                if ip_address not in usernames_targeted:
                    usernames_targeted[ip_address] = set()
                usernames_targeted[ip_address].add(username)

    ip_counts = Counter(attempts)
    flagged = []
    for ip, count in ip_counts.most_common():
        if count >= threshold:
            users = usernames_targeted.get(ip, set())
            print(f"  {ip:18s} {count:4d} attempts — {len(users)} username(s)")
            flagged.append(ip)
    return flagged

if __name__ == "__main__":
    flagged_ips = analyze_auth_log("test_auth.log", threshold=2)
🚨
Result — 192.168.1.100 was flagged as the attacker (3 failed attempts exceeded the threshold of 2). The second IP was not flagged as it only had 1 attempt. Successful run confirmed on synthetic log data.

File Integrity Checker

hashlib sha-256 json os.walk

A "digital fingerprint" tool that uses SHA-256 to baseline-hash all files in a target directory. On subsequent runs, it compares live hashes to the stored baseline and reports any modified or missing files — catching even single-byte tampering via the avalanche effect.

Files are read in 8192-byte chunks so large files don't exhaust memory. Results are saved as JSON for easy review. Run with python integrity_checker.py check to compare against the stored baseline.

Python · integrity_checker.py 
import hashlib, os, json, sys
from datetime import datetime

def hash_file(filepath):
    """SHA-256 'digital fingerprint' — read in 8192-byte chunks."""
    sha256 = hashlib.sha256()
    try:
        with open(filepath, "rb") as f:
            for chunk in iter(lambda: f.read(8192), b""):
                sha256.update(chunk)
        return sha256.hexdigest()
    except (PermissionError, FileNotFoundError) as e:
        return f"ERROR: {e}"

def create_baseline(directories, output_file="baseline.json"):
    """Crawl directories and snapshot every file hash."""
    baseline, count = {}, 0
    for directory in directories:
        for root, _, files in os.walk(directory):
            for filename in files:
                fp = os.path.join(root, filename)
                h = hash_file(fp)
                if not h.startswith("ERROR"):
                    baseline[fp] = {"hash": h, "recorded_at": datetime.now().isoformat()}
                    count += 1
    with open(output_file, "w") as f:
        json.dump(baseline, f, indent=2)
    print(f"Baseline created: {count} files recorded.")
    return baseline

def check_integrity(baseline_file="baseline.json"):
    """Tripwire — compare live hashes against the baseline snapshot."""
    with open(baseline_file) as f:
        baseline = json.load(f)
    modified, missing = [], []
    for fp, record in baseline.items():
        if not os.path.exists(fp):
            missing.append(fp); continue
        current = hash_file(fp)
        if current != record["hash"]:   # Avalanche effect catches single-byte changes
            modified.append({"file": fp,
                              "expected": record["hash"][:16] + "...",
                              "actual": current[:16] + "..."})
    print(f"Files checked: {len(baseline)} | Modified: {len(modified)} | Missing: {len(missing)}")
    return modified, missing

if __name__ == "__main__":
    if len(sys.argv) > 1 and sys.argv[1] == "check":
        check_integrity()
    else:
        create_baseline(["./test_folder"])
        print("Run with 'check' to verify integrity.")
🔒
How it works — Baseline hashes are stored in baseline.json. The avalanche effect of SHA-256 means even a single changed byte produces a completely different hash, making tampering impossible to hide. Run python integrity_checker.py to snapshot, then python integrity_checker.py check to verify.
